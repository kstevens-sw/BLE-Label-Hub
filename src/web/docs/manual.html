Getting started

Open the app in Chrome, Edge, or another Chromium-based browser on a desktop or Android device. Web Bluetooth is the primary connection path for BLE printers. Some printers can also be used over USB when supported by the app. iOS Safari does not support Web Bluetooth.

Fast path: set the connection-type dropdown to match your printer (Bluetooth, Niimbot, or USB), click Connect, then start designing.

Install on mobile: the app is a Progressive Web App — open it in Chrome on Android and choose "Install app" / "Add to Home screen" to run it like a native app, including offline. When installed, the About dialog shows an "Open Desktop Version" link.

1. Connect

Pair the printer from the browser’s device picker.

2. Design

Add text, barcodes, QR codes, shapes, and images to the canvas.

3. Print

Use the print controls to send the finished label to the device.

Workspace basics

The editor is split into three parts:

  • Top controls for connecting, printing, label size, and settings.
  • Center canvas for laying out the label.
  • Right panel for the selected element’s properties and printer options.

Drag elements on the canvas to reposition them. Use the handles to resize or rotate supported items.

The Move tool (cursor icon) at the left of the toolbar is the default select-and-drag mode; click it any time to clear the current selection. The Connect button doubles as Disconnect — once connected it shows "Disconnect", and clicking it drops the printer.

Adding elements

Use the toolbar to add content to the label. Every object can be moved, resized, and styled after placement.

  • Text for names, addresses, prices, or notes.
  • Barcode for Code 128, Code 39, EAN, UPC, and related formats.
  • QR code for URLs, payloads, or short records.
  • Shape for boxes, circles, triangles, and divider lines.
  • Image for logos, symbols, and imported artwork.

Editing properties

Select an element to open its properties. The panel changes based on the item type, but common controls usually include position, size, rotation, alignment, font, and stroke or fill options.

For text fields, you can also insert template fields and expressions so the same design can be reused with different data.

Images: the image properties include brightness, contrast, dithering, and a Black & White button — use it to convert a pasted color image to monochrome so it looks right before printing. Thermal printers always print in black and white, and the dither preview shows the exact output.

Use the keyboard shortcuts shown in the app for undo, redo, duplicate, group, and nudge actions.

Printer setup

Pick the connection type first. The dropdown next to the Connect button (Bluetooth / Niimbot / USB) decides how the app talks to the printer, and it is the only thing that controls this — Print Settings does not change it.
  • Niimbot — for NIIMBOT printers (B1, B21, D11, etc.). Uses the Niimbot protocol library.
  • Bluetooth — for everything else: Jadens JD-468BT, Phomemo, M-series, D-series, P12/A30, TSPL shipping printers.
  • USB — for printers connected over a USB cable, where supported.
Choosing the wrong one is the most common reason a printer pairs in the picker but then fails to connect or print. A Niimbot on "Bluetooth" (or a Jadens on "Niimbot") will not work.

Choose the model that matches your printer in Print Settings. The app uses that profile to select width, DPI, and protocol behavior. The correct default label size is applied automatically when you connect or pick a printer (for example, the Niimbot B1 defaults to 50×30 mm). The connected printer's make, model, and firmware are shown in the bottom-right of the footer.

Printer family Notes
Jadens BLE and TSPL models Use the Jadens profiles and select the exact make/model when possible.
NIIMBOT BLE models Supported through the app’s NIIMBOT transport and printer library integration.
Compatible printer profiles Kept for compatibility with existing hardware layouts and user-created definitions.

If auto-detection does not pick the right model, manually choose the closest match in Print Settings before printing.

Custom printer definitions

You can add or override printer definitions from the printer settings dialog. That is useful when you want a custom width, a different density profile, or a BLE name pattern for automatic detection.

  • Create a new definition when a model is missing from the built-in list.
  • Override an existing definition when your hardware revision behaves differently.
  • Use name patterns to make the app recognize printers automatically on connect.

Troubleshooting

The printer does not appear

Confirm the browser supports Web Bluetooth, keep the printer awake, and verify the model is a BLE-capable device. The picker is filtered to known printer name prefixes; Shift+click Connect to show all Bluetooth devices.

It pairs but will not connect

Bluetooth allows only one connection at a time. If the printer is still linked to its phone app or another device, the browser cannot connect even when you are right next to it. Close that app, toggle Bluetooth off and on, power-cycle the printer, and retry.

Connects but the printer light stays red

The green dot in the app means the Bluetooth link is up. A Niimbot printer also needs its own handshake to finish — watch the status bar for "Printer handshake OK". If it stalls or the link drops, the printer is usually still held by another device.

Wrong connection type

If you see "Not a Niimbot printer" or "no suitable characteristic", the connection dropdown is set to the wrong mode. Use Niimbot for Niimbot printers and Bluetooth for the Jadens and everything else, then reconnect.

Print output is offset

Check width, DPI, and label size in Print Settings. A mismatched printer profile can shift the printed content.

Nothing prints

Confirm the connection type matches the printer (see above), reconnect, verify the selected printer model, and retry with a simple text label first.
